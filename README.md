# Ola-Ride-Booking-Analysis-using-SQL-Power-BI
Analyzed 103K+ ride booking records using SQL to evaluate booking trends, revenue, cancellations, vehicle performance, and customer behavior.
Developed a five-page interactive Power BI dashboard featuring KPIs, revenue analysis, cancellation insights, vehicle-wise performance, and customer/driver rating analysis.
Created SQL queries with filtering, aggregation, grouping, views, and business reporting to generate actionable insights for operational decision-making.
