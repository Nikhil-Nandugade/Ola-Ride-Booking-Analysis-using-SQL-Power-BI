# 🚖 Ola Ride Booking Analysis | Power BI Dashboard

## 📌 Project Overview

The **Ola Ride Booking Analysis** project is an end-to-end Business Intelligence solution developed using **Power BI**, **SQL**, **DAX**, and **Excel**. The project analyzes over **103,000 ride booking records** to uncover business insights related to booking trends, revenue, cancellations, vehicle performance, and customer satisfaction.

The interactive dashboard enables stakeholders to monitor key performance indicators (KPIs), identify operational challenges, and support data-driven business decisions.

---

## 🎯 Project Objectives

- Analyze ride booking performance across different vehicle categories.
- Monitor booking success and cancellation trends.
- Evaluate revenue generation and payment methods.
- Measure customer and driver ratings.
- Build an interactive dashboard for business decision-making.

## 2️⃣ Vehicle Type Analysis

Provides insights into:

## 3️⃣ Revenue Analysis

Tracks financial performance through:

- Total Revenue
- Revenue by Payment Method
- Booking Value Analysis

---

## 4️⃣ Cancellation Analysis

Analyzes cancellation behavior including:

### Customer Cancellation Reasons

- Driver not moving
- Driver asked to cancel
- Change of plans
- Wrong address
- AC not working

### Driver Cancellation Reasons

- Customer not reachable
- Customer requested cancellation
- Vehicle breakdown
- Personal issues
